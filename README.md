## 贡献指南

如果您希望分享某个公司的业务，请按照以下结构创建文件夹和文档：

```
公司名称/
├── README.md                # 公司整体业务概览
├── 部门名称/
│   ├── README.md           # 部门业务概览
│   ├── 业务线1/
│   │   ├── README.md      # 业务线介绍
│   │   └── ...
│   └── ...
└── ...
```

## 内容要求

1. 业务介绍应包含：
   - 业务背景和目标
   - 核心功能
   - 技术架构
   - 业务指标
   - 团队结构

2. 技术内容应包含：
   - 技术栈说明
   - 系统架构图
   - 关键技术选型
   - 性能优化

3. 所有内容需遵守相关法律法规，不得包含敏感信息。
